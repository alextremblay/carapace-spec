# Positional

Define [positional completion](https://carapace-sh.github.io/carapace/carapace/gen/positionalCompletion.html).

```yaml
# yaml-language-server: $schema=https://carapace.sh/schemas/command.json
name: positional
completion:
  positional:
    - [pos1, positional1]
    - [pos2, positional2]
```
